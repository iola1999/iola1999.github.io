# 目前的数据量

- 对局数：65k+

数据确认已清洗过（对局数：详情数 = 1：10），但是目前没有排除有人挂机（评分 0 .0）、投降（败方没有 MVP）的对局。

+ （高端局排位里）蓝、红两方胜率一样吗？

  我们知道农药地图并不是完全对称的，大小龙面朝的方向也不同（会影响开龙、抢龙成功率），目前（截至2019.6.15）的 67496 场对局中，蓝方胜 33169 场，胜率是 49.142%。
  
  这其中因素很多，我对农药理解并不深刻，就不瞎猜了。
  
+ （初步测试）一些英雄对于队友配合的要求

  我找到了这样的一组数据，在阵容中有 鬼谷子 这位英雄时，最终胜、败场数如下：

  |      | 胜利 | 失败 | 总计 |
  | ---- | ---- | ---- | ---- |
  | 五排 | 256  | 243  | 499  |
  | 单排 | 4178 | 4260 | 8438 |
  | 总计 | 4434 | 4503 | 8937 |

  五人排位（可以认为有较好的配合）与单排时，鬼谷子这位英雄胜率确实有些不同，先做出假设 H<sub>0</sub>：鬼谷子队胜负与是否有配合无关，根据独立性检验，

  观测值 $ k = {8937*(256*4260-243*4178)^2 \over 4434*4503*499*8438} = 0.5692 $ 

  这个值小于 $ P(K^2 \geq k) = 0.10 $ （即90%把握）所需的 2.706（，仅约55%把握否认假设H<sub>0</sub>），故不能认为 鬼谷子 对团队配合有较强的要求。

  这与常识有些不符，究其原因，可能是：**五排数据过少**，如果强行五排数据乘以十，那么将能得到观测值约 4.012，这样将能有95%的把握否认假设H<sub>0</sub>。这部分研究暂时还是先放一放。

  任何英雄都需要团队配合，但是注意这里讨论的是特定英雄对团队配合的**需要程度**。
